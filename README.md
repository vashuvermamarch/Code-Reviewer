# Code Reviewer IDE

A high-fidelity, web-based Code Reviewer IDE featuring a Glassmorphism aesthetic, a dark "Dracula" theme, and an interactive 3D Three.js background.

## 🚀 Getting Started

1. Clone the repository.
2. Ensure you have Node.js installed.
3. Run `npm install` in both `FrontEnd` and `BackEnd` directories.
4. Populate `BackEnd/.env` with your API keys (Google Gemini / Groq).
5. Run `launch.bat` (Windows) or `./build.sh` (Linux/macOS) to start the project.

## 🛠️ Features

- **AI Code Review**: Get real-time analysis and scores for your code in multiple languages.
- **Glassmorphic UI**: Premium, modern interface with interactive 3D background.
- **Draggable Windows**: Customize your workspace layout with a floating editor.
- **Dynamic Themes**: Toggle between Dracula and Light modes instantly.

## 📦 Tech Stack

- **Frontend**: React, Vite, Framer Motion, Three.js, Monaco Editor.
- **Backend**: Node.js, Express, Groq SDK, Google Generative AI.
